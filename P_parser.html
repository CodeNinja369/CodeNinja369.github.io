Calculator Recursive Descent Parser in Java

This project allows for the parsing and resolving in bidmas order, of simple integer opperations without negation (+,-,/,*)

Overview:

calc.java acts as the main file and uses "tokenise" to create a list of Token objects. Then parses tokenised list using the parser object method "expr", that outputs a root treenode. Calls resolve method on the root, which will recursively resolve the tree of the input opperation.

Find Repository At:

https://github.com/CodeNinja369/unGodlyCalculatorAscended.java
tnode package

This package holds Treenode and all its subclasses. These classes are designed to make up and resolve the nodes and relations of the parsed tree.

evaluate()

A generic method within the Treenode class and its subclasses. With opperands, it returns the opperands value. With opperators, it returns the evaluation of the opperators left and right with the opperator applied (eg. return this.left.evaluate() * this.right.evaluate(); for multiplication)

Token class

Uses constructor overloading to either store ints or chars, and stores a boolean indicating which the object is.

tokenise()

creates a list of Token objects, storing multi digit numbers or chars

Parser class

stores a list of tokens. Recursively turns those tokens into a tree, as shown in the demonstration above, using the method scn(): which consumes a token, and the exp(), term() and factor() methods, which recursively call eachother to assertain a token's hierarchy in the tree and creating a corresponding Treenode object. Calling the expr method will create the root node and start the parsing process which stores all other tokens as nodes in the tree.
